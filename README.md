## Tecnologias utilizadas

* Dartz
* Dio
* Flutter bloc
* Get It
* Sailor
* Shared Preferences

## Como executar

* Basta utilizar o comando abaixo:

    flutter run

* Caso esteja utilizando algum dispositivo físico ou emulador com arquitetura x64, pode-se executar a versão de release com o comando abaixo:

    flutter run --release
